Comcast - XFinity App

Structure:
The code has been structured and packaged accordingly to reflect different aspects
like views, adapters, model, presenters, interfaces 

Design:
MVP Design Pattern is being used for the app. On launch of the fragment the through the presenter call api request is made to fetch the Characters from the server.

Tablet:
As per the requirement the app is supported on Tablets with the handling of the orientation changes

Product flavors: 
As per the requirement, the app code is shared one to reflect the 2 flavors simpsons and wire characters 
This is done though the Build variants.

Search:
The characters can be searched through the search bar present at the top. The filter happens based on the name and the description of the character.

Error cases:
Error scenarios are handled.
